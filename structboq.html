StructBOQ — IFC Validation System

StructBOQ is not a BOQ generator. It is a quality gate between your model and downstream engineering decisions. It reads your IFC file directly — Revit, ArchiCAD, Tekla, Allplan — and audits every element for classification errors, unit inconsistencies, volume anomalies, and quantity source failures before generating a validated BOQ. Supports IFC2x3, IFC4, IFC4x3.

StructBOQ investigates IFC semantic reliability, quantity traceability, and model assurance across open BIM workflows.

What Gets Validated
  • Element classification — footings, slabs, columns, beams, proxies
  • Quantity source — BaseQuantities, Pset, or geometric fallback per element
  • Unit consistency — cm², mm², m² conversion traps across all software
  • Reinforcement extraction — bar geometry, designation lookup, ratio fallback
  • Semantic anomalies — GrossVolume corruption, void relationships, composite assemblies
